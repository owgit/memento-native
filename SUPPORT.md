# Support

## Where to ask what

- **Questions / How-to / Troubleshooting:**
  - [GitHub Discussions](https://github.com/owgit/memento-native/discussions)
- **Confirmed bug reports:**
  - [GitHub Issues](https://github.com/owgit/memento-native/issues)
- **Feature proposals:**
  - Start in Discussions (`Ideas`) or open a Feature Request issue

## Before opening an issue

Please include:

- Memento version
- macOS version
- Reproduction steps
- Expected vs actual behavior
- Relevant logs/screenshots

## Recommended Discussions categories

- `Q&A`
- `Ideas`
- `Troubleshooting`
- `Announcements`
